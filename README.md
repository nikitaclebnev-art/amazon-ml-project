# Amazon Product Rating Prediction

Этот проект посвящён анализу данных о товарах Amazon и применению методов машинного обучения для предсказания того, будет ли товар высоко оценён. В рамках проекта был использован датасет товаров с Amazon, включающий характеристики товаров, цены, скидки, количество отзывов и рейтинги.

## Цели проекта

Цель проекта — создать модель машинного обучения, которая будет прогнозировать, получит ли товар высокий рейтинг (4.0 и выше) на основе его характеристик.

Задачи проекта:
- Провести исследовательский анализ данных (EDA).
- Очистить и подготовить данные для модели.
- Построить и настроить несколько моделей машинного обучения.
- Оценить производительность моделей и выбрать лучшую.

## Датасет

Для выполнения проекта использован **датасет товаров Amazon**, который можно скачать с [Kaggle](https://www.kaggle.com). Датасет содержит 1465 строк и 16 признаков, включая:
- `product_id`: Идентификатор товара.
- `product_name`: Название товара.
- `category`: Категория товара.
- `actual_price`: Исходная цена товара.
- `discounted_price`: Цена товара со скидкой.
- `rating`: Рейтинг товара.
- `rating_count`: Количество отзывов.
- и другие.
